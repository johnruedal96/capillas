# Skills de OpenCode

Esta carpeta contiene skills personalizados para OpenCode — instrucciones
especializadas que el agente carga automáticamente según la tarea.

## ¿Cómo crear un skill?

1. Crear una subcarpeta con el nombre del skill (ej. `widget-dev/`)
2. Crear `SKILL.md` usando `template/SKILL.md` como base
3. (Opcional) Agregar archivos de referencia en la misma carpeta
